## 系統簡介

### 背景與動機

大學生在準備考試時，常需要反覆練習歷屆考古題。然而，考古題來源分散（學長姐傳承、社團資料、網路搜尋），缺乏統一的整理與分類平台。學生往往花費大量時間尋找特定科目、特定教授或特定年度的考古題，效率低落。

**考古題收藏系統** 旨在提供一個集中管理、分類明確、搜尋便利的考古題資料庫，讓大學生能快速找到所需的歷屆試題，提升備考效率。

### 目標用戶

- **所有大學生**：需要查找、收藏與練習考古題的在校學生
- 特別適用於有期中考、期末考複習需求的學生

### 核心價值

| 價值       | 說明                                           |
| ---------- | ---------------------------------------------- |
| 集中管理   | 將散落各處的考古題集中到單一平台               |
| 快速查找   | 透過科目、教授、年度等多維度分類，快速定位試題 |
| 共享貢獻   | 學生可自行上傳考古題，建立共享的知識庫         |
| 高效備考   | 減少找題目的時間，專注在練習與學習上           |

## 核心功能

### 📖 功能一：科目分類管理

依照不同科目分類瀏覽考古題，快速找到正在修習的課程的歷屆試題。

- 支援新增 / 編輯 / 刪除科目分類
- 每個科目顯示該科目下的考古題數量
- 科目列表可依名稱排序

### 📅 功能二：歷屆年度區分

依照年度與學期篩選考古題，找到特定學期的試題來練習。

- 每份考古題標記所屬學年度與學期（如：113-1、112-2）
- 支援依年度排序與篩選
- 可區分期中考、期末考、小考等考試類型

### 👨‍🏫 功能三：教授分類

按照出題教授篩選考古題，針對目前任課教授的出題風格做準備。

- 支援新增 / 編輯教授資訊
- 每位教授關聯其所屬科目
- 可查看特定教授出過的所有考古題

### ✏️ 功能四：考古題新增與編輯

上傳或新增考古題內容，與其他同學分享手邊的歷屆試題。

- 填寫考古題標題、科目、教授、學年度、學期、考試類型
- 支援以純文字方式輸入題目內容
- 支援編輯與刪除已新增的考古題

### 🔍 功能五：關鍵字搜尋

透過關鍵字搜尋考古題，快速找到包含特定主題或題型的試題。

- 支援依標題、內容進行全文搜尋
- 搜尋結果可再依科目、教授、年度進行篩選

## 系統截圖

### 首頁
系統首頁顯示統計數據（科目數、教授數、考古題數），提供「上傳考古題」和「搜尋考古題」的快速入口，以及最新上傳的考古題列表。

### 科目列表頁
以卡片式顯示所有科目，可點擊進入查看該科目下的所有考古題。

### 新增考古題頁面
提供完整的表單，可填寫標題、科目、教授、學年度、學期、考試類型與題目內容。

## 如何啟動

### 1. 安裝 Flask

```bash
pip install flask
```

### 2. 啟動伺服器

```bash
python app.py
```

### 3. 開啟瀏覽器

打開瀏覽器，前往 `http://127.0.0.1:5000` 即可看到考古題收藏系統。
