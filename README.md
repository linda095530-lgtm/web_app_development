# 考古題收藏系統

這是一個集中管理、分類明確、搜尋便利的考古題資料庫，讓大學生能快速找到所需的歷屆試題，提升備考效率。

## 系統簡介

大學生在準備考試時，常需要反覆練習歷屆考古題。然而，考古題來源分散（學長姐傳承、社團資料、網路搜尋），缺乏統一的整理與分類平台。學生往往花費大量時間尋找特定科目、特定教授或特定年度的考古題，效率低落。

本系統解決了這個問題，將散落各處的考古題集中到單一平台，並且支援多維度分類（科目、教授、年度等）以及關鍵字搜尋。目標用戶為所有大學生，特別適用於有期中考、期末考複習需求的學生。

---

## 核心功能

- **考古題管理：** 學生可自行上傳、編輯與刪除考古題，建立共享的知識庫。
- **科目分類：** 依照不同科目分類瀏覽考古題。
- **教授分類：** 按照出題教授篩選考古題。
- **歷屆年度區分：** 依照學年度與學期（如：113-1、112-2）篩選考古題，並區分考試類型（期中、期末、小考）。
- **關鍵字搜尋：** 支援依標題、內容進行全文搜尋，並可再依科目、教授、年度進行複合篩選。
